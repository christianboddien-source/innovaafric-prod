1. Getting Started

1.1 Base URL

https://innovaafric-api-production.up.railway.app

1.2 Authentication

All protected endpoints require a JWT Bearer token in the Authorization header.

Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

1.3 Access Levels

LevelRolePermissions
Level 1UserOwn profile, own transactions, own wallets, KYC submit
Level 2StaffView all users, view transactions, basic admin operations
Level 3AdminBan/unban users, KYC management, notifications, compliance
Level 4SuperadminFull access: API keys, payroll, system config, all endpoints

1.4 Rate Limits

Rate Limiting
Default: 100 requests/minute per IP. API key partners may have custom limits configured via the Admin Panel. Exceeded limits return HTTP 429 with a Retry-After header.

1.5 Response Format

// Success
{ "success": true, "data": { ... }, "message": "Operation successful" }

// Error
{ "success": false, "error": "Error description", "code": "ERROR_CODE" }

8. Error Codes
HTTP CodeNameDescriptionExample
400Bad RequestInvalid request body or missing required fieldsMissing email in register
401UnauthorizedMissing or invalid JWT tokenNo Authorization header
403ForbiddenInsufficient access levelLevel 1 accessing admin endpoint
404Not FoundResource does not existUser ID not found
422UnprocessableValidation error on dataInvalid currency code
429Too Many RequestsRate limit exceededOver 100 req/min
500Internal Server ErrorUnexpected server errorDatabase connection failed
Error Response Format
{ "success": false, "error": "Human-readable message", "code": "MACHINE_READABLE_CODE", "details": { ... } }
