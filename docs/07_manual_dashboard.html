🇪🇸Español

El Dashboard es la pantalla principal del panel de administración. Muestra un resumen en tiempo real de los indicadores clave de rendimiento (KPIs) del ecosistema INNOVAAFRIC: número total de usuarios registrados, volumen acumulado de wallets en múltiples divisas, cantidad de transacciones procesadas e ingresos generados por comisiones y servicios.

Incluye gráficas de actividad por país, un feed de actividad reciente con las últimas operaciones del sistema, y accesos rápidos a las acciones más frecuentes del administrador. Los KPIs se actualizan automáticamente al cargar la página conectándose al backend de Railway y a Supabase.

Cómo usar

  1. Al iniciar sesión, el Dashboard es la primera pantalla que aparece.
  2. Lee los 4 indicadores KPI en la parte superior: usuarios, volumen, transacciones, ingresos.
  3. Consulta las barras de distribución por país para ver dónde están tus usuarios.
  4. Desplázate hacia abajo para ver el feed de actividad reciente y las alertas del sistema.
  5. Usa los botones de acción rápida para acceder directamente a KYC pendiente, soporte o nuevos registros.

El módulo de Wallets muestra los saldos multi-divisa de todos los usuarios del sistema. Cada usuario tiene un wallet con cuatro bolsillos de divisas: EUR (Euro), USD (Dólar), XAF (Franco CFA de África Central) y XOF (Franco CFA de África Occidental). Desde este módulo el administrador puede realizar ajustes manuales de saldo (crédito o débito) con justificación, exportar todos los datos a CSV para reportes y auditorías, y ver el historial de movimientos de un wallet específico.

  4. Usa "Exportar CSV" para descargar todos los saldos para auditoría.
  5. Haz clic en el nombre de un usuario para ver el historial completo de su wallet.
