Inicio Rápido · Quick Start · Démarrage Rapide
Hacer tu primera llamada a la API en menos de 5 minutos
🔑
Obtener credenciales de API
🇬🇧 Get API credentials · 🇫🇷 Obtenir les identifiants API
🇪🇸 Español

Para integrar con la API de INNOVAAFRIC necesitas una API Key generada desde el panel de administración. Hay dos entornos disponibles.

  1. Inicia sesión en el panel de administración
  2. Ve a Integraciones → Hub → Claves API
  3. Crea una nueva API Key con los permisos necesarios
  4. Copia y guarda la clave — solo se muestra una vez
  5. Usa el entorno sandbox para pruebas y production para producción
🇬🇧 English

To integrate with the INNOVAAFRIC API you need an API Key generated from the admin panel. Two environments are available.

  1. Log in to the administration panel
  2. Go to Integrations → Hub → API Keys
  3. Create a new API Key with the required permissions
  4. Copy and save the key — it is only shown once
  5. Use sandbox environment for testing and production for live
🇫🇷 Français

Pour intégrer l'API INNOVAAFRIC, vous avez besoin d'une Clé API générée depuis le panneau d'administration. Deux environnements sont disponibles.

  1. Connectez-vous au panneau d'administration

  3. Créez une nouvelle clé API avec les permissions nécessaires
  4. Copiez et sauvegardez la clé — elle n'est affichée qu'une seule fois
  5. Utilisez l'environnement sandbox pour les tests et production pour la production

Configurar y recibir webhooks
🇬🇧 Configure and receive webhooks · 🇫🇷 Configurer et recevoir des webhooks
🇪🇸 Español

Los webhooks envían notificaciones automáticas a tu servidor cuando ocurren eventos en INNOVAAFRIC. Tu endpoint debe responder con HTTP 200 en menos de 5 segundos.

  1. Ve al panel admin → Integraciones → Hub
  2. Registra la URL de tu endpoint HTTPS
  3. Selecciona los eventos que quieres recibir
  4. Guarda el Webhook Secret para verificar la firma
  5. Verifica la firma HMAC-SHA256 en cada request recibido
🇬🇧 English

Webhooks send automatic notifications to your server when events occur in INNOVAAFRIC. Your endpoint must respond with HTTP 200 within 5 seconds.

  1. Go to admin panel → Integrations → Hub
  2. Register your HTTPS endpoint URL
  3. Select the events you want to receive
  4. Save the Webhook Secret to verify the signature
  5. Verify the HMAC-SHA256 signature on each received request
🇫🇷 Français

Les webhooks envoient des notifications automatiques à votre serveur lorsque des événements se produisent dans INNOVAAFRIC. Votre endpoint doit répondre avec HTTP 200 en moins de 5 secondes.

  1. Allez dans le panneau admin → Intégrations → Hub
  2. Enregistrez l'URL de votre endpoint HTTPS
  3. Sélectionnez les événements que vous souhaitez recevoir
  4. Sauvegardez le Webhook Secret pour vérifier la signature
  5. Vérifiez la signature HMAC-SHA256 sur chaque requête reçue
// Payload de ejemplo — transfer.completed { "event": "transfer.completed", "timestamp": "2025-06-04T14:32:00Z", "data": { "transferId": "txn_abc123", "status": "completed", "amount": 100, "amountReceived": 63947, "recipientPhone": "+237612345678", "completedAt": "2025-06-04T14:31:55Z" } } // Verificar firma (Node.js) const crypto = require('crypto'); const signature = req.headers['x-innovaafric-signature']; const expected = crypto .createHmac('sha256', WEBHOOK_SECRET) .update(JSON.stringify(req.body)) .digest('hex'); if (signature !== `sha256=${expected}`) throw 'Invalid signature';

Ejemplos de Código · Code Examples · Exemples de Code
Implementaciones de referencia en los lenguajes más usados
🟨
JavaScript / Node.js
// Instalar: npm install node-fetch const fetch = require('node-fetch'); const INNOVA_API = 'https://api.innovaafric.com/api'; const API_KEY = process.env.INNOVAAFRIC_API_KEY; // Crear transferencia async function createTransfer(data) { const res = await fetch(`${INNOVA_API}/transfers`, { method: 'POST', headers: { 'Authorization': `Bearer ${API_KEY}`, 'Content-Type': 'application/json' }, body: JSON.stringify(data) }); if (!res.ok) throw await res.json(); return await res.json(); } // Obtener tasa de cambio async function getRate(from, to, amount) { const params = new URLSearchParams({ from, to, amount }); const res = await fetch(`${INNOVA_API}/exchange-rates/simulate?${params}`); return await res.json(); }

Python
# Instalar: pip install requests import requests, os INNOVA_API = "https://api.innovaafric.com/api" API_KEY = os.environ["INNOVAAFRIC_API_KEY"] HEADERS = { "Authorization": f"Bearer {API_KEY}", "Content-Type": "application/json" } # Crear transferencia def create_transfer(amount, to_currency, recipient_phone): payload = { "amount": amount, "fromCurrency": "EUR", "toCurrency": to_currency, "recipientPhone": recipient_phone, "deliveryMethod": "mtn_momo" } r = requests.post(f"{INNOVA_API}/transfers", json=payload, headers=HEADERS) r.raise_for_status() return r.json() # Verificar estado def get_transfer(transfer_id): r = requests.get(f"{INNOVA_API}/transfers/{transfer_id}", headers=HEADERS) return r.json()

PHP
$INNOVA_API = 'https://api.innovaafric.com/api'; $API_KEY = getenv('INNOVAAFRIC_API_KEY'); function innovaRequest($method, $path, $data = null) { global $INNOVA_API, $API_KEY; $ch = curl_init($INNOVA_API . $path); curl_setopt_array($ch, [ CURLOPT_RETURNTRANSFER => true, CURLOPT_CUSTOMREQUEST => $method, CURLOPT_HTTPHEADER => [ "Authorization: Bearer " . $API_KEY, "Content-Type: application/json" ], CURLOPT_POSTFIELDS => $data ? json_encode($data) : null ]); $result = json_decode(curl_exec($ch), true); curl_close($ch); return $result; } // Ejemplo: crear transferencia $transfer = innovaRequest('POST', '/transfers', [ 'amount' => 50, 'fromCurrency' => 'EUR', 'toCurrency' => 'XAF', 'recipientPhone' => '+237612345678', 'deliveryMethod' => 'orange_money' ]);
